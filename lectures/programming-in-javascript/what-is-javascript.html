A programming language is a text written that a computer can read and convert it to bits (0/1) that it understands. Bits are the computer's language.
Unfortunately, people don't understand bits, but they can understand code.

There are many programming languages that exist. And every programming language has its own syntax.
Some languages have simmilar syntax.
For example, C++, Javascript and Java have simmilar syntax.

We can divide programming languages into two main groups:

  1. Object-oriented programming languages
    It is called like that because it has objects and objects are used for having some data and properties organized inside data holders - objects.
  2. Not object-oriented programming languages
    These programming languages only have writing lines of code after lines of code... and then executing.
I personally prefer object-oriented programming.
But that doesn't mean that you need to prefer it, too.

So, Javascript is an object-oriented programming language, because it has all those objects about changing elements on our website and computing stuff.

Javascript was created by Brendan Eich in 1995.
To read more about it, go to this site.

Javascript was first called Livescript, but because it had very simmilar syntax with Java (like classes, declaring objects, try/catch, etc...), it was renamed to Javascript.

Except...
Javascript is a programming language for a browser/interpreter that can execute it, but it cannot be used to compile and run it on your local computer.
Javascript is for a browser, for interactive apps, for Scratch projects and other.
