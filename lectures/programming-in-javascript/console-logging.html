Let's start with Javascript.
The simplest way to see what we are doing, or see what we have done and the errors, is by logging. Logging can be done by writing text in the console.
The console is a way to log things, make errors, do warnings and things like that.
You can open it by clicking on the F12 button.

But, how can we write inside the console?
We can do this by the console.log() method.

The console.log() method is a method of the class console.

It needs one parameter: that's anything. You can put something like undefined, a number, a string, or even null!
Here's an example code: 


// Logging text
console.log("Hello world!");
console.log("I am writing inside the console.");

// Logging numbers
console.log(123);
console.log(20 * 20);

// Logging special values
console.log(undefined);
console.log(Infinity);
console.log(null);

// Logging objects
// When logging objects, you can browse them in the console (like a hierarchy)
console.log(console);
console.log(window);
console.log(document);
console.log({firstname: "Mihoko", lastname: "Ferguson"});
console.log(["banana", "mango", "papaya", "orange"]);

// Logging function return values
console.log(parseInt("492101"));
console.log(parseFloat("69.420"));
console.log("Hello world!".toUpperCase());
console.log(console.log(console.log("Here console.log will be called multiple times in a line!")));
